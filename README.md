<h1>Select Every N Components for Maya</h1>

<p>
A lightweight Maya tool that allows you to quickly select every Nth Face, Edge, or Vertex from an existing component selection.
Useful for modeling, retopology, pattern creation, paneling, procedural workflows, and repetitive selection tasks.
</p>

<h2>Features</h2>

<ul>
    <li>Select every Nth face from a face loop.</li>
    <li>Select every Nth edge using Maya's edge ring functionality.</li>
    <li>Select every Nth vertex from the current selection.</li>
    <li>Configurable step value and offset.</li>
    <li>Face loop validation to prevent invalid selections.</li>
    <li>Simple and lightweight interface.</li>
</ul>

<h2>How to Use</h2>

<ol>
    <li>Select a component type (Faces, Edges, or Vertices).</li>
    <li>Adjust the <strong>Step</strong> value.</li>
    <li>Optionally adjust the <strong>Offset</strong> value.</li>
    <li>Press the corresponding selection button.</li>
</ol>

<h3>Examples</h3>

<ul>
    <li>Step = 2 → Select one component, skip one.</li>
    <li>Step = 3 → Select one component, skip two.</li>
    <li>Offset = 1 → Start from the second component.</li>
    <li>Offset = 2 → Start from the third component.</li>
</ul>

<h2>Requirements</h2>

<ul>
    <li>Autodesk Maya</li>
    <li>Python (Maya embedded Python)</li>
</ul>

<h1>Seleccionar Cada N Componentes para Maya</h1>

<p>
Una herramienta ligera para Maya que permite seleccionar rápidamente cada N Cara, Arista o Vértice a partir de una selección existente.
Resulta útil para modelado, retopología, creación de patrones, panelado, flujos de trabajo procedurales y tareas repetitivas de selección.
</p>

<h2>Características</h2>

<ul>
    <li>Selecciona cada N cara de un loop de caras.</li>
    <li>Selecciona cada N arista utilizando la funcionalidad de Edge Ring de Maya.</li>
    <li>Selecciona cada N vértice de la selección actual.</li>
    <li>Valor de intervalo y desplazamiento configurables.</li>
    <li>Validación de loops de caras para evitar selecciones inválidas.</li>
    <li>Interfaz sencilla y ligera.</li>
</ul>

<h2>Cómo usarlo</h2>

<ol>
    <li>Selecciona un tipo de componente (Caras, Aristas o Vértices).</li>
    <li>Ajusta el valor de <strong>Step</strong>.</li>
    <li>Opcionalmente ajusta el valor de <strong>Offset</strong>.</li>
    <li>Pulsa el botón correspondiente.</li>
</ol>

<h3>Ejemplos</h3>

<ul>
    <li>Step = 2 → Selecciona un componente y salta uno.</li>
    <li>Step = 3 → Selecciona un componente y salta dos.</li>
    <li>Offset = 1 → Comienza desde el segundo componente.</li>
    <li>Offset = 2 → Comienza desde el tercer componente.</li>
</ul>

<h2>Requisitos</h2>

<ul>
    <li>Autodesk Maya</li>
    <li>Python (incluido en Maya)</li>
</ul>
